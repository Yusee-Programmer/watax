<h1 align="center">watax</h1>

<p align="center">
  <em>A fast, ergonomic web framework for <a href="https://github.com/Yusee-Programmer/tauraro">Tauraro</a> —
  Python-like handlers, Rust-class performance, and automatic memory management.</em>
</p>

---

## Quick start

```tauraro
from watax import App
from std.net.http_server import HttpConn
from std.encoding.json import JsonValue

def root(c: HttpConn):
    c.send_text(200, "Hello, World!")

def greet(c: HttpConn):
    mut name = c.request.get_param("name")
    mut o = JsonValue.init_object()
    o.read().obj_set("hello", JsonValue.init_str(name))
    c.send_json_value(200, o)        # watax owns + frees the whole tree

def main():
    mut app = App.new()
    app = app.get("/", root)
    app = app.get("/greet/:name", greet)
    app.listen_reactor_pool("127.0.0.1", 8080, 4)
```

```sh
taupkg build && ./watax          # or: tauraroc src/main.tr -o watax && ./watax
```

> Method-chaining across lines isn't supported by the parser yet, so reassign
> the app on each builder call: `app = app.get(...)`.

## Features

- **Routing** with path params (`/users/:id`), wildcards, route groups, and mounts.
- **Responses**: text · HTML · JSON · `JsonValue` · templates · files · redirects ·
  status-only, plus **chunked streaming** and **Server-Sent Events**.
- **Requests**: path/query/form params, JSON bodies, cookies, headers, and
  **multipart file uploads**.
- **Templates** via [templa](https://github.com/Yusee-Programmer/templa)
  (`{% extends %}` / `{% block %}` inheritance + filters).
- **Static files** with content-type detection, caching, and SPA fallback.
- **WebSockets** (RFC 6455) and **TLS** (behind `-DTAURARO_TLS_OPENSSL`).
- **Middleware**: before/after hooks, CORS, rate limiting, request logging,
  signed-cookie sessions.
- **Servers**: thread-per-connection and an epoll/kqueue/WSAPoll **reactor pool**.
- **Zero manual memory management in handlers** — the framework and Tauraro's
  auto-drop free every per-request allocation.

## Performance

watax matches Rust (axum/hyper) on throughput while using **less memory**, and is
~2.4× faster + ~11× leaner than FastAPI. Numbers and methodology in
[`benchmarks/`](benchmarks/) → [`benchmarks/results.md`](benchmarks/results.md)
(regenerated by CI).

The reference app in [`example/app/`](example/app/) shows templates, static files,
form CRUD, a JSON API, CORS, and centralized 404 handling end-to-end.

<p align="center">
  <strong>watax</strong> is built on the idea that a web framework should be
  <em>simple to write</em> and <em>cheap to run</em>.
</p>

<p align="center">
  You write plain handler functions; the compiler and framework carry the
  memory-management burden, so your code stays Python-clean while serving at
  native speed with a flat, predictable memory footprint.
</p>
