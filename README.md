# watax

A fast, ergonomic web framework for [Tauraro](https://github.com/Yusee-Programmer/tauraro)
— Python-like handler code with Rust-class performance and **automatic memory
management**: your handlers never call `free`/`dispose`.

```tauraro
from watax import App
from std.net.http_server import HttpConn
from std.encoding.json import JsonValue

def root(c: HttpConn):
    c.send_text(200, "Hello, World!")

def greet(c: HttpConn):
    mut name = c.request.get_param("name")
    mut o = JsonValue.init_object()
    o.read().obj_set("hello", JsonValue.init_str(name))
    c.send_json_value(200, o)        # watax owns + frees the whole tree

def main():
    mut app = App.new()
    app = app.get("/", root)
    app = app.get("/greet/:name", greet)
    app.listen_reactor_pool("127.0.0.1", 8080, 4)
```

## Features

- **Routing** with path params (`/users/:id`) and wildcards.
- **Responses**: text / HTML / JSON / templates / files / status-only, plus
  chunked streaming and Server-Sent Events.
- **JSON**: build a `JsonValue` and hand it to `send_json_value` — watax
  serializes, sends, and frees the entire tree for you.
- **Templates** via [templa](https://github.com/Yusee-Programmer/templa)
  (`{% extends %}` / `{% block %}` inheritance, filters).
- **Static files**, **multipart uploads**, **WebSockets** (RFC 6455),
  **TLS** (behind `-DTAURARO_TLS_OPENSSL`).
- **Middleware** (before/after), CORS, request logging, cookies.
- **Servers**: thread-per-connection (`listen_threaded`) and an epoll/kqueue/
  WSAPoll **reactor pool** (`listen_reactor_pool`).
- **No manual memory management in user code** — the framework + Tauraro's
  auto-drop free every per-request allocation.

## Memory & performance

watax matches Rust (axum/hyper) on throughput and **uses less memory**, while
being ~2.4× faster and ~11× leaner than FastAPI. See
[`benchmarks/`](benchmarks/) and the auto-generated
[`benchmarks/results.md`](benchmarks/results.md) (regenerated by CI).

## Getting started

```sh
taupkg build        # or: tauraroc src/main.tr -o watax
./watax
```

The reference application lives in [`example/app/`](example/app/) — a
server-rendered + JSON-API notes app demonstrating templates, static files,
CRUD, CORS, and centralized 404 handling.
