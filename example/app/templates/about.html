About this app

This page is part of watax-notes, the reference example application for the watax web framework. It demonstrates a small but complete setup:

  • Server-rendered HTML pages via templa templates with a shared layout
  • Static CSS and JavaScript served from /static
  • Form handling for creating, completing and deleting notes
  • A JSON API under /api/notes guarded by CORS middleware
  • Request logging and configuration via environment variables
